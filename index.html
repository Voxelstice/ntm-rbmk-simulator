Welcome to NTM RBMK Simulator.
This site is meant to simulate the mechanics
of NTM's RBMK in a browser.

If you are unfamiliar with NTM's RBMKs, check out the wiki page.
Building a RBMK is not that easy, you need to think carefully.

To start placing, click on any existing block and
click on any tile.


You destroyed the RBMK.
Congratulations.


The site's source code is public. GitHub
This simulator by default uses Large Turbines and Auxiliary Cooling Towers for the boilers.
